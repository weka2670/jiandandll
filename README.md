# tlbb-server

#### 架设步骤
第一步
```
yum install -y wget git vim && \
cd ~ && git clone https://gitee.com/QQ63732956/tlbb-server.git /opt
```
第二步
```
cd /opt && chmod +x install.sh
sh install.sh
```
第三步
```
用WinSCP工具上传天龙八部服务端到home目录下
```
第四步
```
用Xshell工具连接服务器
解包tlbb.tar.gz
命令如下

cd /home
tar xzvf tlbb.tar.gz
```
